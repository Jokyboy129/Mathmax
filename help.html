Rechner Handbuch

Willkommen zur Hilfe. Dieser Rechner unterstützt mathematische Ausdrücke, Analysis, Statistik, Vektorrechnung und das Lösen von Gleichungen.

Eingabe: Sie können Dezimalzahlen sowohl mit Punkt (3.14) als auch mit Komma (3,14) eingeben. Bei Funktionen werden Argumente mit Semikolon (;) getrennt (z.B. min(1; 5)).

Grundrechenarten & Konstanten

Funktion Eingabebeispiel Beschreibung
Grundrechnung 12 + 5 * (3 - 1) Klassische Punkt-vor-Strich-Rechnung.
Potenz 2^3 oder 2**3 Zwei hoch drei.
Wurzel sqrt(16) Quadratwurzel.
n-te Wurzel root(27; 3) Die 3. Wurzel aus 27. (Syntax: root(Zahl; Grad))
Logarithmus ln(e), lg(100), log(8; 2) Natürlicher (ln), 10er (lg) oder Basis-Logarithmus (log(Zahl; Basis)).
Konstanten pi, e Kreiszahl Pi und Eulersche Zahl.

Analysis (Differentialrechnung)

Funktionen zur Berechnung von Ableitungen.

Befehl Eingabebeispiel Beschreibung
nderive nderive(x^2; 2; 1) Berechnet den numerischen Wert der Ableitung an einer Stelle.
Syntax: nderive(Funktion; Position; Grad).
Beispiel: 1. Ableitung von x² an der Stelle x=2 ist 4.
algs algs(3x^3; 1) Gibt die Ableitungsfunktion symbolisch zurück.
Syntax: algs(Funktion; Grad).
Beispiel: 1. Ableitung von 3x³ ist 9x².

📊 Statistik & Kombinatorik

Alle statistischen Funktionen können sowohl einzelne Zahlen als auch Vektoren verarbeiten.

Befehl Eingabebeispiel Beschreibung
min / max min(4; 1; 8) Gibt den kleinsten oder größten Wert zurück.

Vektorrechnung

Vektoren werden in eckigen Klammern geschrieben und mit Strichpunkt getrennt: [x; y; z].

Definition [1; 2; 3] Erstellt einen Vektor.
Skalarprodukt dot([1; 0]; [0; 1]) Berechnet das Skalarprodukt (Zahl).
Kreuzprodukt cross([1;0;0]; [0;1;0]) Berechnet das Kreuzprodukt (nur bei 3D-Vektoren).
Länge (Betrag) len([3; 4]) Berechnet die Länge des Vektors.

Gleichungslöser

Lösen von Gleichungen (nsolve)

Findet exakte oder numerische Lösungen für eine Variable. Findet auch mehrere Lösungen (z. B. bei quadratischen Gleichungen).

Syntax: nsolve(Gleichung)

  • nsolve(x^2 - 4 = 0) → Findet alle Lösungen für x.
  • nsolve(x^2 = 4) → Alternativschreibweise.

Lineare Gleichungssysteme (lsolve)

Löst ein System aus linearen Gleichungen exakt. Trennen Sie Gleichungen mit einem Strichpunkt.

Syntax: lsolve(Gl1; Gl2; ...)

  • Beispiel: lsolve(x + y = 5; x - y = 1)

Eigene Funktionen

Sie können eigene Funktionen definieren, anzeigen und löschen.

  • Definieren: def f(x): x^2 + 2 (Danach können Sie f(3) rechnen)
  • Anzeigen: showdef (Listet alle gespeicherten Funktionen auf)
  • Löschen: deldef f (Löscht 'f') oder deldef (Löscht alle)

Calculator Manual

Welcome to the help section. This calculator supports mathematical expressions, calculus, statistics, vector calculations, and equation solving.

Input: You can enter decimals using a dot (3.14) or a comma (3,14). For functions with multiple arguments, separate them with a semicolon (;) (e.g., min(1; 5)).

Functions for calculating derivatives.

Command Example Input Description

Syntax: algs(Function; Degree).

Vector Calculations

Vectors are written in square brackets and separated by semicolons: [x; y; z].

Definition [1; 2; 3] Creates a vector.
Dot Product dot([1; 0]; [0; 1]) Calculates the dot product (scalar).
Cross Product cross([1;0;0]; [0;1;0]) Calculates the cross product (3D vectors only).
Length (Magnitude) len([3; 4]) Calculates the length of the vector.

Equation Solvers

Solving Equations (nsolve)

Finds exact or numerical solutions for a variable. Can also find multiple solutions (e.g., for quadratic equations).

Syntax: nsolve(Equation)

  • nsolve(x^2 - 4 = 0) → Finds all solutions for x.
  • nsolve(x^2 = 4) → Alternative notation.

Systems of Linear Equations (lsolve)

Solves a system of linear equations exactly. Separate the equations with a semicolon.

Syntax: lsolve(Eq1; Eq2; ...)

  • Example: lsolve(x + y = 5; x - y = 1)

Custom Functions

You can define, display, and delete your own functions.

  • Define: def f(x): x^2 + 2 (Afterward, you can calculate f(3))
  • Display: showdef (Lists all saved functions)
  • Delete: deldef f (Deletes 'f') or deldef (Deletes all)
