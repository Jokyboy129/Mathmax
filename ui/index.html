Geben Sie Ihren mathematischen Ausdruck in das Eingabefeld ein und drücken Sie die Eingabetaste (Enter).

  • Grundrechenarten: +, -, *, /
  • Gleichungen lösen: nsolve(x^2 - 4 = 0)
  • Gleichungssysteme lösen: lsolve(x + y = 10; x - y = 2)
  • Funktionen definieren: def f(x): x^2 + 2
  • Funktionen anzeigen: showdef
